## ✨ 关键模块详解

### 🔹 1. 去公众号源 + 内容去重

```python
# 伪代码：资讯过滤逻辑
def filter_articles(raw_articles):
    # 排除公众号域名
    exclude_domains = ['mp.weixin.qq.com', 'weixin.qq.com']
    
    # 标题+摘要指纹去重 (防止多渠道转载)
    seen_hashes = load_cache('last_7days_hashes')
    
    filtered = []
    for art in raw_articles:
        if any(domain in art['url'] for domain in exclude_domains):
            continue
        if md5(art['title']+art['summary']) in seen_hashes:
            continue
        filtered.append(art)
    
    # 按权威性+时效性评分，取Top 5
    return rank_and_select(filtered, top_n=5)
```

### 🔹 2. 人工编辑风格 Prompt 模板（核心！）

> 参考知乎/掘金实战经验 [[11]][[14]]，这是去除"AI味"的关键：

```markdown
# AGENTS.md - 您的专属编辑人设

## 🎭 角色设定
你是一位有5年科技媒体经验的编辑"小科"，说话带点个人情绪，喜欢用"说实话""讲真""这波"等口语，偶尔吐槽，但专业度在线。

## ✍️ 写作规则
1. 禁用词：首先/其次/综上所述/值得注意的是/总而言之
2. 句式：70%段落≤3句话，多用短句、问句、感叹句
3. 语气：像跟朋友聊天，可适当用"😅""👉""🔥"等表情
4. 结构：不写小标题，用自然过渡；每篇结尾加一句个人看法
5. 细节：提到公司/产品时加一句背景（如"这家刚融了$50M的初创..."）

## 📝 输出格式
【标题】不超过25字，带数字/反差/悬念
【正文】300-500字，分3-4段，首段抓眼球，末段留思考
【标签】#AI #大模型 #行业观察（选2-3个）

## 🎯 示例风格
❌ AI味：首先，Gemini 2.5 Pro在多项基准测试中表现优异...
✅ 人味：谷歌这次真拼了，Gemini 2.5 Pro一上来就把上下文拉到100万...说实话，这参数党看了直呼过瘾，但实际用起来咋样？我试了俩小时，发现...
```

## 🚀 快速启动指南（30分钟部署）

### Step 1：准备环境
```bash
# 1. 安装核心工具
npm install -g @wenyan-md/cli
brew install imagemagick  # 图片处理

# 2. 获取微信凭证（服务号需认证）
# 公众号后台 → 开发 → 基本配置 → AppID/AppSecret

# 3. 配置环境变量
echo 'export WECHAT_APP_ID="xxx"' >> ~/.zshrc
echo 'export WECHAT_APP_SECRET="xxx"' >> ~/.zshrc
source ~/.zshrc
```

### Step 2：创建风格规范
在项目根目录创建 `AGENTS.md`，填入上方的人设模板。

### Step 3：编写主流程脚本
```bash
#!/bin/bash
# daily_brief.sh - 每日简报生成脚本

# 1. 抓取资讯 (调用您的爬虫或RSS聚合)
python fetch_tech_news.py --exclude-wechat --top 10 > raw.json

# 2. AI编辑 (调用Claude API + AGENTS.md)
claude code --agents ./AGENTS.md \
  "基于raw.json，选3-5条写成人工风格简报，输出article.md"

# 3. 生成封面 (可选)
# 用KIE API或Unsplash随机科技图 + ImageMagick裁剪
magick cover.jpg -crop 900x383+0+0 cover_wechat.jpg

# 4. 发布到草稿箱
wenyan publish -f "./article.md" --cover "./cover_wechat.jpg"

# 5. 通知您审核
curl -X POST "https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=YOUR_KEY" \
  -d '{"msgtype":"text","text":{"content":"📰 今日简报已生成，请审核草稿箱"}}'
```

### Step 4：设置定时任务
```bash
# Linux/macOS cron (每天9点)
crontab -e
# 添加：
0 9 * * * /path/to/daily_brief.sh >> /var/log/tech_brief.log 2>&1

# 或用 GitHub Actions (免费+可视)
# .github/workflows/daily.yml
on:
  schedule: [{ cron: "0 1 * * *" }]  # UTC 1am = 北京时间9am
jobs:
  publish:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - run: bash daily_brief.sh  # 需配置secrets
```

## 🛡️ 避坑指南（实战经验）

| 坑点 | 现象 | 解决方案 |
|------|------|----------|
| **微信IP白名单** [[1]] | `40164 invalid ip` | 用 `curl ipinfo.io/ip` 查出口IP，固定代理节点 |
| **AppSecret泄露** | 接口权限异常 | 环境变量存储 + 定期重置 + 最小权限原则 |
| **封面图裁剪** | 显示不全/变形 | 严格按 900×383px (2.35:1) 裁剪，用ImageMagick批量处理 |
| **AI味残留** | 读者反馈"像机器写的" | 迭代AGENTS.md + 人工抽检 + 加入个人经历/吐槽 |
| **内容同质化** | 多号撞车同一新闻 | 加"个人解读"权重 + 小众信源（如arXiv最新论文） |
